git clone
git branch
git branch -a
git remote -v
git add -- lets us add new files to be tracked by git to our repo
git commit -a -- saves all of our changes on the local computer into a thing called a commit. A commit is a set of changes.
git commit file1 file2 ... -- commits only specific files to our local branch
git push -- this will send our local commits to our default remote git service (eg github)
git pull -- checks the remote serves for new commits; if there are this command will fetch (git fetch) them and merge the new changes with our local branch (git merge)
